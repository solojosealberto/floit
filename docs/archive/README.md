# Docs Archive

Contiene documentos historicos o de baja rotacion que no deben competir con la fuente operativa de estado.

## Reglas

- No usar estos archivos para decidir estado de sprint/release.
- Si un documento vuelve a ser operativo, moverlo fuera de `docs/archive/`.
- Mantener referencias desde `docs/index.md` cuando sea util para contexto historico.
