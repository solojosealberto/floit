# QueGym final brand assets

Paquete listo para integrar la marca QueGym en website, PWA, social previews y app icon.

## Paleta aplicada

- Ink: `#0B0F0E`
- Ink 2: `#151B19`
- Green principal: `#17C964`
- Mint: `#9FE6C3`
- Mist: `#EAF8F1`
- White: `#FFFFFF`

## Rutas recomendadas para Next.js / public

Copia el contenido de esta carpeta dentro de `public/` manteniendo la estructura:

- `/logos/png/logo-horizontal-transparent.png`
- `/logos/png/symbol-transparent.png`
- `/icons/favicon/favicon.ico`
- `/icons/favicon/favicon.svg`
- `/icons/app/apple-touch-icon.png`
- `/icons/pwa/android-chrome-192x192.png`
- `/icons/pwa/android-chrome-512x512.png`
- `/social/og-image.png`
- `/site.webmanifest`

## Nota de fidelidad

Los PNG y SVG wrappers fueron extraídos desde la imagen de marca aprobada para preservar el diseño visual seleccionado. Los SVG incluidos embeben el raster exacto del logo, por lo que mantienen la apariencia aprobada. Para producción avanzada o impresión, conviene vectorizar manualmente en Figma/Illustrator usando este paquete como referencia maestra.
